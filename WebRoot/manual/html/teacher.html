经管实验教学系统使用说明

  • 教师
  • 学生

 /  添加课程

  1. 所有操作如右图所示：
  2. 登录后，首先点击"上机任务"
  3. 在显示出的二级菜单中点击"实验课程"，将会显示出所有的实验课程
  4. 在罗列出来的实验课程中(也可在搜索框内输入你想添加的课程名称),找到你要添加的课程的行，点击改行"添加课程"按钮，即可添加成功，添加成功的课程可在"我的任务"中查看

 /  发布任务1

  1. 所有操作如右图所示：
  2. 在显示出的二级菜单中点击"我的任务"，将列出自己所添加的所有的任务
  3. 点击你想查看任务该行的"发布任务"按钮，将会弹出一个框上传实验指导书，如下一页图片所示。

 /  发布任务2

  1. 所有操作如右图所示：
  2. 首先在弹出的框中，输入实验指导书的名称
  3. 其次，点击"浏览"按钮选择自己要上传的文档
  4. 最后点击"开始上传"

 /  查看任务

  1. 所有操作如右图所示：
  2. 在显示出的二级菜单中点击"我的任务"，将列出自己所添加的所有的课程
  3. 点击你想查看任务该行的"+"按钮，将会展开此课程的所有任务
  4. 点击实验指导书名称将下载此实验指导书，也可以删除此课程、删除任务、查看学生作业、修改任务，具体操作如右图所示

 /  查看学生作业

  1. 所有操作如右图所示：
  2. 在显示出的二级菜单中点击"我的任务"，将列出自己所添加的所有的课程
  3. 点击你想查看任务该行的"+"按钮，将会展开此课程的所有任务
  4. 点击你想查看作业的行的"查看作业"按钮，页面将会如右图所示
  5. 选中你想下载作业的学生姓名，也可全选，选中后点击"下载"按钮,将你选中的作业下载至本地进行查看（可通过在搜索框内输入你想下载作业的学生名字快速找到该作业）

 /  上传资料

  1. 所有操作如右图所示：
  2. 首先在一级菜单栏中点击"学习资料"
  3. 再点击"资料上传"按钮，将会弹出一个弹出层上传资料
  4. 在弹出层首先点击"添加文件"，选择要上传的文件
  5. 添加完文件后，点击"开始上传"，将添加的文件上传到服务器端，最后关闭弹出层，添加的资料可在"我的资料"和"所有资料"中找到

 /  上传软件

  1. 所有操作如右图所示：
  2. 首先在一级菜单栏中点击"软件下载"
  3. 再点击"软件上传"按钮，将会弹出一个弹出层
  4. 在弹出层首先点击"添加文件"，选择要上传的文件
  5. 添加完文件后，点击"开始上传"，将添加的文件上传到服务器端，最后关闭弹出层，添加的软件可在"我的软件"和"所有软件"中找到

 /  学生认证

  1. 所有操作如右图所示：
  2. 在页面右上角个人信息旁找到橘黄色链接"学生信息"，点击此链接，将会弹出一个弹出层
  3. 在弹出的弹出层中将会显示所有学生的信息
  4. 对在线注册的学生进行认证（可通过在搜索框中输入学生姓名快速找到你想认证的学生）
